# Age-Calculator
The Age Calculator App is a simple and interactive web application that allows users to calculate their exact age based on the date of birth. Built using HTML, CSS, and JavaScript, this app displays the user's age in years, months, and days.

Features:
Enter your date of birth
Calculate accurate age (Years, Months, Days)
❌ Prevent future dates using validation
Clean and responsive design
Real-time age calculation using JavaScript

Technology:
HTML – To create the structure and form input
CSS – For attractive styling and layout
JavaScript – To calculate the age and handle validations
